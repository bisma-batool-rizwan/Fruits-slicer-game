# Fruits-slicer-game 
# 🍉 Fruit Slicer Game

A fun and interactive browser-based Fruit Slicer Game developed using **HTML**, **CSS**, and **JavaScript**. Players slice falling fruits to earn points while trying to achieve the highest possible score. The game features smooth animations, responsive gameplay, and an engaging user experience.

## 🌐 Live Demo

**Play Here:**
https://fruits-slicer-game.netlify.app/

---

## ✨ Features

* 🍎 Randomly falling fruits
* ✂️ Smooth fruit slicing animations
* 🏆 Real-time score tracking
* ❤️ Lives and Game Over system
* 🎮 Simple and engaging gameplay
* 📱 Responsive design for different screen sizes
* ⚡ Built with clean and lightweight code

---

## 🛠️ Built With

* **HTML5** – Structure
* **CSS3** – Styling and animations
* **JavaScript (ES6)** – Game logic and interactivity
